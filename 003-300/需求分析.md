# 需求分析

1. 红外检测到人时，根据光照传感器监测的光强自动调节LED亮度（光强高LED越暗）。
2. 红外未检测到人时，无论光强如何，LED均关闭。
3. 支持通过蓝牙远程控制LED开关。
4. 显示屏显示当前光感强度和时间。

# 器件表

- 主控芯片：ESP32-WROOM-32D
- PIR传感器：HC-SR501
- 光敏传感器：GL5528光敏电阻
- LED光源：WS2812B（RGB可编程LED）
- 电源模块：TP4056锂电池管理模块
- 显示屏（型号未定）
